Are there differences in activity patterns between weekdays and weekends?

First we need a new colomn in the new data that indicates the weekday of a date. We can write a function here in order to use the sapply function.

convert_weekday <- function(date) {
  if(weekdays(as.Date(date)) %in% c("Saturday", "Sunday")) {
    return("weekend")
  }
  else {
    return("weekday")
  }
}

Now we can easily create a new column.

new.data$weekday <- as.factor(sapply(new.data$date, convert_weekday))
